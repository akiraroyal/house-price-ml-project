# House Price Prediction (Machine Learning)

This project builds an end-to-end machine learning model to predict home sale prices using the Kaggle Ames Housing Dataset

##  Problem Statement
Predict the final sale price of homes based on property features such as size, location, quality, and amenities.


## Skills Demonstrated
Data cleaning, Feature engineering, Categorical encoding, Log transformation, Regression modeling, Model evaluation (RMSE), Visual diagnostics
